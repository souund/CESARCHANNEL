Perfil de Artista

Ocho anos descifrando la pista. El mismo criterio en cada ciudad, en cada salon, en cada atmosfera. El lugar cambia. El oido no.

La seleccion musical no se apoya en formulas ni playlists previsibles. Cada set se construye desde la lectura del espacio, la energia de los invitados y el tempo real del evento.

FORMATOS DE SERVICIO

Configuraciones de performance y produccion adaptadas a la identidad de cada evento.

Formato 01

DJ Set

La ejecucion central del set con control estetico de cabina y criterio musical completo. Pensado para venues o montajes que ya cuentan con infraestructura tecnica propia.

  • Cesar Channel en controles
  • Cabina visual frontal
  • Sesion adaptable al formato del evento
Formato 02

DJ Set + Audio

Una solucion mas completa para espacios que requieren refuerzo sonoro profesional ademas del set. Equilibrio entre performance, claridad tecnica y presencia visual.

  • Performance + cabina
  • Sistema de audio profesional
  • Configuracion segun venue y aforo
Formato 03

DJ Set + Audio + Iluminacion

Una propuesta escenica mas inmersiva donde el sonido y la iluminacion trabajan en conjunto para transformar la percepcion del espacio y sostener la atmosfera durante toda la noche.

  • Audio profesional
  • Iluminacion para pista y ambiente
  • Montaje coordinado con la identidad del evento
Formato 04

Produccion Integral + Host

La version mas completa del servicio: direccion de ambiente, produccion tecnica, presencia de host y acompanamiento del protocolo clave para eventos que necesitan conduccion y ritmo.

  • Audio + iluminacion + cabina
  • Host y apoyo de protocolo
  • Coordinacion general del bloque musical

RIDER TECNICO

Control Booth

  • Mesa de control estable e independiente (minimo 1.50 m).
  • Alimentacion electrica regulada a pie de cabina.
  • Proteccion contra intemperie o rayos solares directos.

Audio & Monitoreo

  • Sistema PA adecuado a las dimensiones del espacio.
  • Monitoreo local en cabina regulable de forma independiente.
  • Linea de senal limpia XLR directo a sistema central.

Hospitalidad

  • Zona de resguardo segura para equipamiento tecnico secundario.
  • Acceso a hidratacion constante durante montaje y ejecucion.
  • Coordinacion directa con el staff tecnico del venue.
